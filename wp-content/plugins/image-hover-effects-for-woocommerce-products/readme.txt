Allow user to display there woocommerce products with css3 animation effects any where they want.
== Description ==


Image Hover effects for WooCommerce products is a collection of pure css3 animations for images.You just need to select the cateogry of products from plugin settings and plugin will display all products of that category with the animation effects. This would increase interactivity of user to your products.User can also add to cart product using ajax system set by plugin mean user don't need to go to product page to add product to cart and amazing is that there would be no page reload when product added to cart in this way user can add multiple products to cart at a time. You can display your products in grid or slider.It gives you full control over changing your product content designs.It is really simple and fast to set up in few seconds. It is really userful for the end user who is not having any programing knowledge.
